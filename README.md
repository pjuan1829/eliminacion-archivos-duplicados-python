# Sistema de Eliminación de Archivos Duplicados en Python

Este proyecto implementa una herramienta en Python que detecta archivos duplicados dentro de una carpeta comparando su contenido mediante hashes generados con la librería `hashlib`. El sistema conserva la versión más reciente de cada archivo y elimina automáticamente las copias antiguas, optimizando el espacio y mejorando la organización.

## Características
- Detección de archivos duplicados por contenido, no por nombre.
- Eliminación automática de versiones antiguas.
- Conservación garantizada de la copia más reciente.
- Código simple, legible y basado en librerías estándar de Python.
